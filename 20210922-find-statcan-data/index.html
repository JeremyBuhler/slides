About Statistics Canada

Serving Canada with high-quality statistical information that matters

"Statistics Canada produces statistics that help Canadians better understand their country - its population, resources, economy, society and culture."

from https://statcan.gc.ca/about/about

A few words about Statistics Candidate. Its mission is to collect and provide data about Canada's population, resources, economy, society, and so on.

But serving Canada does not necessarily mean making the data available to everyone. In fact, withholding some of this data from public use is part of StatCan's service: preserving the privacy of people who respond to the census or to surveys, for example.

Statistics Canada Open Licence

https://statcan.gc.ca/reference/licence

Grants permission to

  • use
  • reproduce
  • publish
  • freely distribute
  • sell
But there is a subset of data that Statistics Canada does release for public use under the term of their Open License. This is quite a permissive license.
So what kind of data is out there and how does it get used?
Chances are you've come across news stories like these that cite Statistics Canada as their source. This story is a fairly high level overview that provides some recent statistics about employment in Canada, including the overall unemployment rate of 7.1%.
This is great if the news story has exactly what you're interested in, but what we see in this story are descriptive statistics: a summary that's easy to understand but that doesn't leave you much flexibility. What if you wanted to know how unemployment rates differ by province or by age group?

Public Use Microdata Files (PUMFs)

Public data modified to protect respondent privacy

  • Some variables suppressed
  • Some variables aggregated
  • May only be a sample of responses

Where is this stuff?

Statistics Canada Data search https://www150.statcan.gc.ca/n1/en/type/data

Abacus Open Data collection https://abacus.library.ubc.ca/dataverse/open

Steps for Statistics Canada Data search. make sure window is sized so subjects appear on left, not top

  • Orient, expand subject list
  • Filter by subject "Labour", note large number of results
  • Search keywords "employment"
  • Switch to "Tables" tab, select "Labour force characteristics by province..." in new tab
  • Scroll down, show geography parts
  • Point out the 7.1% unemployment rate from article
  • Scroll to bottom, show source of data (Labour Force Survey)
  • Return to search results
  • Switch to PUMF tab and select LFS (second in list)
  • Note data availability, indicated by links (older data not available)
  • Two reasons to go to Abacus: PUMF exists but is not accessible from site, or PUMF is there but want more documentation and related files, or data in a different format (LFS is only in CSV on Statistics Canada site, but SPSS file in Abacus)

After Statistics Canada, ask about presentation speed

Steps for Abacus Open Data collection

  • Orient, show filter options
  • Note that most recently updated datasets appear first
  • Open Labour Force Survey 2021, orient
  • Note large number of files, note the tags. Try not to scroll past the tag chooser so you don't need to scroll back up.
  • Explain importance of documentation, open first file, show TOC
  • Return to record, apply August tag to filter
  • Show August files, show tab file (what to look for to find the data, the tags and the file descriptions
  • Download the Aug LFS .sav file, note the license (which you mentioned earlier)
  • Open in SPSS, go to variable view
  • Switch to data view, make it clear that this freely available data includes the individual responses from the nearly 87K people who completed the August 2021 Labour Force Survey
  • Pause for questions
  • If no questions, return to Abacus open data and search for CCHS, browse files and point out documentation, etc.
